JOBSEA — Job Portal Web Application
JOBSEA is a full-stack job portal website built for job seekers across India. It allows users to browse and apply for jobs in 4 major categories — Education, IT, Sales, and Government sectors.
🔧 Tech Stack

Frontend: React + Vite + Tailwind CSS
Backend: Java Spring Boot
Database: MySQL
Authentication: Google OAuth2 + Phone OTP

✨ Features

Google Sign-In authentication
Phone number OTP login
4 job categories — Education, IT Jobs, Sales Jobs, Govt Jobs
Admin panel to add, edit and delete job listings
External application links for each job
Job search by keyword
Featured jobs section
Animated hero section with 3D effects
Contact form — messages saved directly to database
Admin dashboard to view all user messages
Fully responsive design

🗂️ Project Structure
JOBSEA/
├── backend/   → Spring Boot REST API
└── frontend/  → React Vite Application
🚀 How to Run Locally

Create MySQL database named jobportal
Update application.properties with your MySQL credentials
Run Spring Boot backend on port 8080
Run React frontend on port 5173
Login with Google and start exploring!
